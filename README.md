# Keylogger

A basic Python keylogger program that records and 
logs keystrokes to a file for educational purposes.

## 📌 Task
This project was completed as Task 4 of my Cyber 
Security Internship at **SkillCraft Technology**.

## ⌨️ About Keylogger
A keylogger is a program that records every key 
pressed on a keyboard. In cybersecurity, understanding 
keyloggers helps professionals learn how attackers 
steal sensitive information and how to defend against 
such threats.

## ⚠️ Disclaimer
This keylogger is built strictly for **educational 
purposes** as part of a Cyber Security Internship. 
It should only be used on your own device. Unauthorized 
use of keyloggers is illegal and unethical.

## ⚙️ Features
- Records all keystrokes in real time
- Logs special keys (Enter, Space, Shift etc.)
- Saves keystrokes with timestamp to a log file
- Press ESC to stop the keylogger

## 🛠️ Technologies Used
- Python
- Pynput Library

## ▶️ How to Run
1. Install the required library: pip install pynput
2. Run the script: keylog.py
3. Start typing — all keystrokes will be recorded
4. Press ESC to stop
5. Open keylog.txt to see recorded keystrokes
